## Dota2Guide

![Dota2Guide LOGO](https://github.com/noisay/Dota2Guide/blob/master/app/src/main/res/mipmap-mdpi/ic_launcher.png)

DOTA2新手指导,android app

DOTA2中英雄资料查看软件，所有数据离线存储，无联网权限，节省流量。
